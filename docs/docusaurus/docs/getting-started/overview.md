HVE Core accelerates software development by providing pre-built GitHub Copilot customizations for every phase of the development lifecycle. This section covers what you need to get started.

## What You Get

When you install HVE Core, your Copilot environment gains access to:

- Specialized agents for research, planning, architecture, and code review
- Prompt templates for common development tasks
- Instructions that automatically enforce your team's coding standards
- Skills that bundle documentation with executable scripts

## Prerequisites

- Visual Studio Code with GitHub Copilot extension
- GitHub Copilot Chat enabled
- A GitHub repository where you want to use HVE Core

Continue to [Installation](installation) to set up HVE Core in your project.
